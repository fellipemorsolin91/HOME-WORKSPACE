# Ferramentas com passo manual

## Navicat

Motivo do manual: software comercial com download/licenca proprietaria.

Passos:

1. Baixe o instalador Linux no site oficial.
2. Instale conforme instrucoes do fornecedor.
3. Salve neste repositorio qualquer arquivo de configuracao exportavel.

## SQL Developer

Motivo do manual: download da Oracle normalmente exige login/aceite.

Passos:

1. Baixe o zip do SQL Developer na Oracle.
2. Extraia em um caminho padrao (ex.: ~/apps/sqldeveloper).
3. Crie um atalho .desktop, se desejar.
4. Exporte preferncias/conexoes quando possivel e versione no repo.
